## Installation

1. Clone the repo

```sh
git clone --branch module-9 https://github.com/codiologist/ostad-vue-mastering.git
```

2. Go to directory
```sh
cd ostad-vue-mastering
```
```sh
cd vue3-blog-assignment
```

3. Install packagees with
```sh
npm install
```

4. Run project by
```sh
npm run dev
```
